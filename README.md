# PLC 통신



## 프로젝트 소개

현장에서 **PLC** 그리고 장비들끼리 많도 다양한 데이터들을 주고 받습니다.

이번 프로젝트는 **PLC**간의 데이터, 그리고  **Invertor** 의 데이터 값을  **ethernet 통신**을 하여 값을 주고 받습니다.



## 프로젝트 개요

**PLC1  <-> PLC2** 간의 **모드버스 TCP,  사용자정의(485) 프로토콜**을 사용하여 통신.

**Invertor** 의 결괏값을 XGT, 모드버스 TCP 프로토콜을 사용하여 통신.


## 사용기술

**XG-5000**

**LS electric plc**
